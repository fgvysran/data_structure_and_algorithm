3. 二分

##### 3.1 有序数组差某个数是否存在

```java
public boolean binary_search(int[]arr,int l,int r,int num){(l~r-1)
    r--;
    while(l <= r){
        int mid = (l + r) >> 1;
        if(arr[mid] > num){
            r = mid - 1;  
        }else if(arr[mid] < num){
            l = mid + 1;
        }else{
            return true;
        }    
    }return false;
}
```

##### 3.2 有序查找>= x最左侧

```java
public int binary_search(int[]arr,int l,int r,int num){(l~r-1)
```

##### 3.2 局部最小值问题(arr无序，相邻两个数不相等，求一个局部最小)

```java
public int binary_search(int[]arr,int l,int r){//[l~r]
    if(arr[l] < arr[l + 1])return 0;
    if(arr[r] < arr[r - 1])return r;
    while(l <= r){
        int mid = (l + r) >> 1;
        if(arr[mid] < arr[mid + 1] && arr[mid] < arr[mid - 1]){
            return mid;
        }else if(arr[mid] < arr[mid + 1]){
            r = mid - 1;
        }else{
            l = mid + 1;
        }
    }return l;
}
```
