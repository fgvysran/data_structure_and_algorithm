## 1. 异或运算

1. a ^ a = 0

2. a ^ 0 = a

3. a ^ b = b ^ a

4. a ^ b ^ c = a ^ (b ^ c)

所以交换两个数可以：a = a ^ b; b = a ^ b;a = a ^ b;(前提a 和 b 在内存里是两块独立区域)

##### 1.1 习题：

###### 习题一：一个数组，只有一个数出现奇数次，其他偶数次，求该数

```
初始化eor=0,eor与数组中每个数异或，最后的eor为答案
```

###### 习题二：一个数组，只有两个数出现奇数次，求这两个数

```
a最右侧的1：(~a + 1) & a

假设是a,b。eor=a ^ b.

所以 a != b && eor != 0.(eor一定某一位不为零，假设地7位为1，a 和 b 在第七位上一定不同)（eor2 = 0,只异或第七位为1的数）

eor2 和eor2 ^ eor 为答案
```
