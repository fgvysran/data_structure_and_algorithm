# 快速排序

快速排序利用 **分治思想**：

1. **选一个基准数（pivot）**

2. **划分数组**：把数组分成
   
   - `< pivot` 的区间
   
   - `= pivot` 的区间
   
   - `> pivot` 的区间

3. **递归排序左右两边**（小于区间、大于区间），中间等于区间不用再排。

 **荷兰国旗问题（Dutch National Flag）分区法**。

```java
public static void quick_sort(int[]arr,int l,int r){
        if(l < r){
            int start = l;
            swap(arr,l + (int)(Math.random() * ( r - l + 1)),r);
            int x = l - 1,y = r;//小于，大于区域
            while(l < y){
                if(arr[l] < arr[r]){
                    swap(arr,++x,l++);
                }else if(arr[l] > arr[r]){
                    swap(arr,--y,l);
                }else{
                    l++;
                }
            }
            swap(arr,r,y++);
            quick_sort(arr,start,x);
            quick_sort(arr,y,r);
        }
    }
```



## 📌 时间复杂度

- 平均：`O(n log n)`

- 最坏：`O(n²)`（但随机 pivot 能大幅降低概率）

- 空间复杂度：`O(log n)`（递归栈）
